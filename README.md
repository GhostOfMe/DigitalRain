# digital_rain

![Screenshot](https://github.com/GhostOfMe/DigitalRain/blob/master/data/img/screenshot.png)

Rust implementation of foating text from movie "The Matrix".

# Installation

Via Cargo:
```sh
git clone https://github.com/GhostOfMe/digital_rain.git
cd ./digital_rain
cargo install --path .
digital_rain
```
